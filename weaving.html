Weaving Projects

Weaving is the crafting discipline I've been doing the longest. My sister taught me the basics, and I got hooked! I had so much fun learning different techniques and styles, and I still find it so exciting how many different projects you can create with the same set of fundamental knots! Here's some of my favorite projects from over the years.
